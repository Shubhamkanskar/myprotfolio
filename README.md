## 🚀 Quick Start

1. **Clone Repository**
   ```bash
   git clone https://github.com/Shubhamkanskar/myportfolio.git
   cd myportfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **View Website**
   ```
   Open http://localhost:3000
   ```
